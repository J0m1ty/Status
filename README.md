# A comprehensive status page

This status page allows me to manage all my services on my server, from viewing their uptimes to restarting individual services (or the entire server). View the live page at https://status.jomity.net.

![status jomity net_](https://github.com/J0m1ty/Status/assets/60862595/fbf57fb8-7056-41fa-9a66-e3091ea31116)

# Development information

This project was largely me experimenting with React.js (+ MUI). Now I can explore Next.js or a different UI framework. This project also uses Express.js for the backend, and Nginx for handling the proxy-pass.

# Potential updates

I'd like to move this to an entierly seperate hardware, so it can also track reliability, downtime, and latency on the main server.
